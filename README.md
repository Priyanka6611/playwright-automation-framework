# Playwright Automation Framework

Enterprise-style Playwright automation framework demonstrating UI testing, API testing, Page Object Model and CI/CD concepts using C#.

## Features

* UI Automation Testing
* API Testing
* Page Object Model (POM)
* Data-Driven Testing
* Test Reporting
* CI/CD Ready Structure

## Technology Stack

* Playwright
* C#
* .NET
* REST APIs
* GitHub Actions

## Framework Structure

```text
Tests/
Pages/
Utilities/
TestData/
Reports/
```

## Purpose

This repository demonstrates modern test automation practices and framework design concepts commonly used in enterprise environments.
